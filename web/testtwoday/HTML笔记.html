  4. 什么是HTML？ HTML是用来写网页的？

    • 网页：指浏览器用来显示互联网数据的一个工具。互联网数据有纯文本、超链接、图片、音频、视频、列表之类的。

    • 网页的本质：是什么构成了网页。一个网页是HTML代码的体现。

    • html: 超文本标记语言(hyper-text Markup language)

      • 超文本：不仅仅是文本，本质上指的是网页中能够显示的数据格式：纯文本、超链接、图片、按钮、视频、音频、输入框。而HTML代码就用来显示超文本。
      • 标记/标签：在HTML中，一个超文本内容会通过一个标签来体现。标签指利用<>和字母的组合来显示超文本。
        • HTML中拥有固定数量的标签。不支持自定义标签。
        • 每一种标签对应一种超文本。比如标签用来显示超链接。标签用来展示图片。
        • 标签一般语法：
          • <标签名>要显示内容
          • 例如: 百度一下
        • 语言：计算机语言。即完成人和计算机之间的沟通。我们可以使用HTML来编写网页，同时浏览器可以读懂HTML代码并解析成网页内容。
        • 属性
          • 每个标签都有属于自己的属性或者叫特征。每个标签的属性数量是固定的。
          • 语法：<标签名 属性名="属性值">>
            • 例如：<a href='https://www.baidu.com'>百度

  8. 标准HTML页面

    1. 标准HTML页面具有一些基础的标签，用于设置网页的标题、网页的一些配置等操作。
    2. <!DOCTYPE html>
      1. 告诉浏览器该页面的HTML版本。浏览器就会以对应HTML版本的语法来解析HTML页面。该声明是HTML5版本的声明。如果HTML页面有以前HTML版本的语法，那么HTML5是向后兼容的，所以也会正常显示。
    3. <html>标签：
      1. 每个HTML页面都有且只要一个标签。HTML页面内容都应该在该标签下面
    4. <head>: 用于完成网页的一些配置。比如说设置网页的标题，或者指定网页的字符编码格式等。
    5. <body>：网页主体。理论上 我们要在网页中显示的所有内容都是放在该标签之中。
    6. <title>:专门写在head标签里，用于指定网页的标题
    7. <meta>:配置标签，用于完成页面的一些常规配置，比如字符编码、或者页面关键字、缓存等。一般网页就直接写即可

文件路径

  1. 路径：是指某个文件在某个目录的位置描述。路径分为两种：绝对路径、相对路径
  2. 绝对路径：
    1. 指某个文件的唯一位置的描述。比如说蜗牛学院成都校区：四川省成都市高新区孵化园园区5号楼3层301.
    2. 在程序中，绝对地址一般指网络地址(以HTTP或HTTPs开头)或者以系统盘符开头（c:/）的地址.
  3. 相对地址：指某个文件根据某个参考文件夹而相对的位置描述。即从当前目录出发，其他文件所在的位置。
    1. 例子：朋友来找你吃饭，他现在在孵化园地铁口，你打电话说：从地铁a扣出来，往前走100米，然后右转200米，我在那等你。
    2. 程序中，找到eye.png 我们会从所编辑的文件所在目录出发（该目录就是参考目录，右称为当前目录），先退回到上一级目录(第一周),再找子目录(images)，在images中找到eye.png。
    3. 相对地址语法：
      1. 上一级目录: ../
      2. 子目录: 子目录/
      3. 例子： ../images/eye.png

表格

  1. 表格标签

    1. : 表示创建一个表格。具体几行几列由tr和td的数量来决定
      1. border：设置表格的边框。以数字为单位。默认为0.即默认没有边框。
      2. width：设置表格的宽度。以像素为单位
      3. height:设置表格的宽度。以像素为单位
      4. align:设置表格本身相对于页面的水平对齐方式。有left 、center、right
      5. cellspacing：设置单元格之间的距离。默认为1，以数字为单位。
      6. cellpadding：设置单元格边框跟其内容之间的距离。有默认值。
      7. bgcolor：设置表格的背景颜色
    2. :表示行。一个tr表示表格中的一行。
      1. align：设置一行中数据的水平对齐方式。有left 、center、right。
      2. height：设置该行的高度
      3. bgcolor:设置一行背景颜色
    3. :表示单元格。一个td表示一行中的一个单元格。
      1. align：设置该单元格数据的对齐方式。有left、right、center。
      2. width：设置单元格的宽度
      3. height：设置单元格的高度
      4. colspan:用于跨列合并。比如colspan="3"，表示合并3个单元格.
      5. rowspan:用于跨行合并。比如rowspan="3".表示合并3个单元格。
      6. bgcolor:设置单元格的颜色
    4. :表示一个标题列。本质上就是一个特殊的td。th中的文字默认都是粗体。

  2. 注意事项

    1. 在HTML中，表格默认是没有边框以及宽高的，所以我们要看到效果，那么就必须设置table的边框以及宽高。
    2. 在给表格填充数据时，如果有某行没有数据，那么该行的空间会被其他有数据的行给占领。
    3. 如果每一行都有数据，或者都没有数据，那么表格会吧宽高平均分配到每一行。
    4. 标题行在HTML中有专门的标签来书写，即th。
    5. 表格中，标题行数据是水平垂直居中的。表格的每一行数据都是垂直居中的。
    6. 如果需要实现数据水平居中，就可以设置tr或td的align属性为center。
    7. 如果同一行中tr td都设置了align属性。对于td而言，会使用td里面的align属性。
    8. table中，tr，td、th都可以设置宽高。但要注意的是如果是给tr、td、th设置高度，都会影响整一行的高度。设置td、th宽度时，会影响整个一列
    9. table、tr、td都可以设置bgcolor。即背景颜色。同时几者之间的 覆盖关系为 td会覆盖tr，tr会覆盖table，table会覆盖body。

表单

1. 概念：表单是指专门用于收集用户信息的单子。
2. 表单元素：用于收集用户信息的HTML标签
    1. 输入框
    2. 单选框
    3. 多选框
    4. 下拉菜单
    5. 文本域
    6. 按钮
3. 表单标签
    1. <form></form>：表示一个表单，所有收集信息的表单元素都放在form标签里
    2. <input />:输入框，具体输入的类型由input标签中的type属性来指定
    3. <select></select>: 下拉菜单，菜单中的每个选项需要使用<option>标签
    4. <textarea></textarea>:用于收集用户输入的多行文本
    5. <button></button>:按钮，按钮中的文本直接在中间书写。
4. 表单元素
    1. 输入框
    2. 密码框：
    3. 单选框：<input type='radio'>
        1. 如果需要实现单选，那么需要给input设置一样的name属性。
        2. 如果其中一个选项为默认，那么就给input标签添加checked属性即可。即<input type='radio' checked='checked'>。同时checked可以省略，即<input type='radio' checked>
    4. 多选框：<input type='checkbox'>
        1. 如果多选框表示同一种情景，比如爱好。那么一般都会设置一样的name属性。
        2. 同时支持添加checked属性来实现默认勾选。可以添加多个
    5. 下拉菜单： select
        1. 菜单中的每个选项需要使用option标签
        2. 可以给某个option标签添加selected属性来设置为默认选项。如果没有selected，那么第一个选项是默认选项
    6. 文本域： <textarea></textarea>
        1. rows控制文本域能够输入多少行文字，但不同浏览器效果不一，后期会使用css控制。变相控制文本域的高度。
        2. cols控制一行能够输入多少个文字。变相的是控制文本域的宽度。
        3. 默认情况下，文本域是可以缩放的。后期可以使用css控制。
    7. 按钮
        1. input
            1. <input type='button' value='按钮上显示的文字'>
        2. button
            1. <button>按钮显示的文字</button>
        3. 以下按钮写法具有提交表单的功能
            1. <input type='submit' value=‘提交’>
            2. <button>提交</button>
        4. 当点击提交按钮后，那么会吧数据提交到form标签中action属性所指的页面
